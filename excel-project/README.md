# Excel Sales & Profit Dashboard

This Excel project analyzes product and regional performance using pivot tables, charts, and KPI summaries.

---

## 🔧 Steps Performed

- Cleaned raw dataset (removed blanks, standardized categories)
- Created pivot tables for:
  - Sales by Region
  - Profit by Product
  - Monthly Sales Trend
- Built KPI cards:
  - Total Sales
  - Total Profit
  - Profit Margin %
  - Average Order Value
- Designed a clean, business‑friendly dashboard layout

---

## 📊 Dashboard Highlights

- Region **North** leads in revenue  
- Product **Category A** shows highest profitability  
- Sales peak in **Q2** and **Q4**  
- Profit margin improves when discounting is reduced
